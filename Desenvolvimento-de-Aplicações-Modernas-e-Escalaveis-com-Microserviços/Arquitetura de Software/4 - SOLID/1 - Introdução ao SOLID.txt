Introdução ao SOLID {
        "SOLID é um acrônimo que consolida 5 itens que considerados como boas práticas no mundo do desenvolvimento
        orientado a objetos."

        S  -> SRP = Single Resposibillity Principle

        O -> OCP = Open-Closed Principle

        L -> LSP = Liskov Substitution Principle

        I -> ISP = Interface Segregation Principle

        D -> DIP = Dependecy Inversion Principle
}
